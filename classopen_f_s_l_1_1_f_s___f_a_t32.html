◆ initialize()

int FS_FAT32::initialize ( )

FS_FAT32 intializer.

Initializes Disk driver and FAT32 class. Function checks signatures:

  • Jump code in BPB (0xEB 0x58 0x90)
  • VBR Signature in BPB (0x55 0xAA)
  • FSINFO Signature (0x55 0xAA)

Warns if free cluster amount is unknown (0xFFFFFFFF) Warns if path separator is not given (default="\") Loads FATArea to memory

Returns
int: Number of errors
